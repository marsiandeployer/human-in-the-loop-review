How SimpleReview works on CS-Cart

1

Install the Chrome Extension

Add SimpleReview to Chrome. Auto-detects CS-Cart via <meta name="generator" content="CS-Cart"> and the storefront HTML signatures (Multi-Vendor and B2B editions are recognised automatically).

2

Auto-detect via the generator tag

SimpleReview reads the platform, edition (Free / Plus / Ultimate / Multi-Vendor / B2B), version, and active theme straight from the page. No copy-pasting your install path.

3

Connect Git or SFTP

GitHub, GitLab, Gitea, Bitbucket, or direct SFTP. SimpleReview indexes design/themes/, app/addons/, and the active overrides folder on first connection.

4

Click → describe → get a PR

Hover any block, button, or footer credit. Type "remove this duplicate newsletter" or "rename this checkout label". The agent edits the right .tpl (or addon schema) and opens a PR — with cache:clean and reindex notes in the description if the change touches schemas or layouts.

FAQ

Can I edit my CS-Cart store without a developer?
For most cosmetic and structural edits — yes. Theme templates (design/themes/[theme]/templates/), CSS, language strings, addon overrides. SimpleReview opens a PR for those in seconds. For payment gateway integration, vendor RBAC, and ERP work — pair with Vibers human review.
How does SimpleReview install on CS-Cart and is it on the CS-Cart Marketplace?
Today: install the Chrome Extension and connect your Git repo or give SFTP credentials. The native CS-Cart addon simplereview is in approval at marketplace.cs-cart.com. Once approved: Add-ons → Manage add-ons → upload archive → Install → Settings → SimpleReview → connect Git/SFTP.
Which CS-Cart versions are supported?
CS-Cart 4.x — full support across single-vendor, Multi-Vendor, and B2B editions. Free Edition has limited features but works the same way. Tested on the Responsive theme and popular vendor themes that follow the standard override pattern.
Does it work with CS-Cart Multi-Vendor and B2B?
Yes. Multi-Vendor and Multi-Vendor Plus run on the same theme/addon engine; SimpleReview detects the edition from the storefront generator tag and adapts file paths accordingly. B2B-specific features (corporate pricing, request-a-quote) live in addons under app/addons/ and edit identically.
Can SimpleReview hide the "Powered by CS-Cart" line?
Technically — yes; it lives in design/themes/[theme]/templates/blocks/static_templates/copyright.tpl. Legally — it depends on your CS-Cart license. Free Edition requires the credit; some Plus / Ultimate license tiers have specific clauses about copyright removal. Read your license before merging the PR.
How is this cheaper than a CS-Cart freelancer at $80/hr?
Speed and recurring cost. A freelancer takes 1-3 hours just to spin up a CS-Cart sandbox, find the right addon/override, and validate. The actual edit is often a 3-line template patch. SimpleReview already knows the layout (design/themes/[theme]/templates/, app/addons/[my_addon]/, design/themes/[theme]/templates/overrides/) and opens the PR in seconds. For complex work — custom payment gateway, vendor RBAC, ERP integration — pair with Vibers.
