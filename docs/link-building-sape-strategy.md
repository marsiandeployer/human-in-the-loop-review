## Критерии донора (фильтры в Sape)

### Обязательно:
- Authority Score (AS) / DR ≥ 25
- Трафик сайта ≥ 1 000 визитов/мес
- Тематика: developer tools, SaaS, software engineering, tech, AI tools, open source
- Язык: EN (английский контент, EN-аудитория)
- Индексация: страница в поиске Google

### Запрещено:
- Сайты без трафика (sape-мусор)
- Doorway / affiliate-помойки
- Нетематические площадки (мода, медицина, недвижимость и т.д.)
- Ссылки с одной страницы вместе с 10+ другими арендными ссылками

---

## Стратегия анкоров

Распределение для каждой целевой страницы:

| Тип | Доля | Примеры |
|-----|------|---------|
| Branded | 40% | "Vibers", "Vibers Review", "Vibers Code Review" |
| Naked URL | 20% | https://onout.org/vibers/ |
| Partial match | 25% | "human code review", "code review service" |
| Natural / generic | 15% | "this tool", "learn more", "check it out" |

НЕ спамить exact match ("best ai code review tools") — выглядит неестественно.

---

## Типы площадок — что искать в Sape

По убыванию приоритета:

### Tier A — Максимальный вес
- Dev blogs с реальным трафиком (AS 35+)
- Статьи про code review, AI tools, developer productivity
- Доноры с авторским контентом, не агрегаторы

### Tier B — Хороший вес
- SaaS-каталоги и directories (AS 30+): toolify.ai, saashub.com-подобные
- Tech media / newsletters c EN-аудиторией
- GitHub-экосистема: блоги про open source, DevOps

### Tier C — Только если дёшево и тематично
- Общие IT-блоги (AS 25-30)
- Статьи про AI, machine learning, software quality

---

## Что НЕ покупать

- Ссылки с RU-тематических сайтов на EN-продукт (не релевантная аудитория)
- Сайты с признаками PBN (похожий дизайн, много исходящих ссылок)
- Любые сайты с трафиком < 500/мес — бесполезно

---

## Бюджет и темп

- Старт: 3-5 ссылок/мес (не наращивать резко)
- Средний бюджет на ссылку: $15-50/мес (качественный донор EN)
- Мониторинг: раз в месяц проверять что ссылки активны (Sape Dashboard)

---

## Сравнение с уже имеющимися возможностями

Бесплатные ссылки (из backlink-opportunities.md) — делаем параллельно:
- Tier 1 каталоги: theresanaiforthat.com, g2.com, saashub.com — бесплатно, сделать до начала аренды
- Dev.to, Hashnode, Medium — статьи с ссылками, бесплатно

Arендные ссылки (Sape) нужны для усиления позиций по коммерческим ключам,
где бесплатных ссылок недостаточно.

---

## Первый шаг — чеклист для Sape

1. [ ] Зарегаться на sape.ru как рекламодатель
2. [ ] Создать 5 проектов (по одному на каждую целевую страницу выше)
3. [ ] В каждом проекте: фильтр по тематике "IT/Software/SaaS", язык EN, AS≥25, трафик≥1K
4. [ ] Выбрать 1-2 донора на каждую страницу, не больше
5. [ ] Анкор: branded или partial match (не exact)
6. [ ] Через 4 недели: проверить индексацию ссылок через Google `site:donor.com`
