## Что забрать в наш context

1. **Шаблонная фабрика — рабочий паттерн**, но требует Information Gain. Наш angle: интерактивные демки вместо PDF.
2. **"Alternative to" — низкая конкуренция** для каждого SaaS. Уже взяли 3.
3. **Free-positioning** — наше преимущество перед всеми.
4. **Не уходить в UAT/proofing** — не наша expertise (Google March 2026).
