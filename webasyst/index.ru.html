Что SimpleReview правит на Shop-Script

Smarty-шаблоны в wa-apps/shop/themes/[theme]/
Backend плагины в wa-apps/shop/plugins/[plugin]/
Хуки (actionExample, frontend_*, backend_*)
Языковые файлы в wa-apps/shop/themes/[theme]/locale/[locale]/
CSS/JS в wa-apps/shop/themes/[theme]/css/, js/
Конфиги в wa-config/db.php, wa-config/SystemConfig.class.php
Удаление «Сайт работает на Webasyst» из подвала (с учётом лицензии Free)
Captcha форм через waCaptcha
Совместимость: работает с Webasyst 2.x + Shop-Script 9 и 10. Также применим к другим приложениям Webasyst (CRM, Site, Mailer, Blog) — структура wa-apps/[app]/ единая.

Как SimpleReview работает на Shop-Script

1

Установить расширение

Поставьте SimpleReview из Chrome Web Store. Кликните по иконке расширения, когда открыт ваш магазин на Shop-Script.

2

Auto-detect Webasyst

Платформа определяется автоматически по <meta name="generator" content="Webasyst"> и характерным сигнатурам HTML. SimpleReview сразу подгружает Webasyst-режим.

3

Подключить Git или SFTP

GitHub, GitLab, Gitea, Bitbucket — или прямой SFTP. SimpleReview индексирует wa-apps/, wa-config/, темы и плагины при первом подключении.

4

Кликнуть → получить PR

Кликаете по нужному блоку, описываете правку. Агент находит файл в wa-apps/shop/themes/[theme]/ или плагине, открывает Pull Request с заметкой про php wa.php install [plugin_id], если плагину нужна реинсталляция.

FAQ

Можно ли дорабатывать Shop-Script без программиста?
Для большинства правок темы и контента — да. Smarty-шаблоны в wa-apps/shop/themes/[theme]/, CSS, языковые файлы, хуки. SimpleReview за секунды откроет PR. Для интеграций с 1С, эквайрингом, миграции между версиями — лучше связка с человеческим ревью Vibers.
Как установить SimpleReview на Shop-Script (Webasyst)?
Сейчас: ставите Chrome Extension и подключаете Git-репо или SFTP. В планах: нативный плагин в Webasyst Store (на ревью у Webasyst, обычный цикл 2-6 недель). После публикации — установка через Установщик: php wa.php install simplereview.
Какие версии Webasyst поддерживаются?
Webasyst Framework 2.x + Shop-Script 9 и 10 — полная поддержка. Поддерживаемые темы: default, Hypermarket, Dummy, кастомные темы, следующие конвенции override.
Работает ли с другими приложениями Webasyst — CRM, Site, Mailer, Blog?
Да. Webasyst — это framework, на котором работают Shop-Script, Webasyst CRM, Site, Mailer, Blog, Helpdesk. Все используют единую структуру wa-apps/[app]/, поэтому SimpleReview одинаково работает с темами, плагинами и хуками любого из этих приложений.
Безопасно ли использовать на проде Shop-Script?
SimpleReview никогда не пушит в живую ветку напрямую. Каждое изменение — Pull Request, который вы ревьюите и мержите. Worst case — вы отклонили PR, прод не пострадал. Для критичных правок рекомендуем staging-копию (всё ещё проще, чем платить 3000 ₽/час).
Чем это дешевле фрилансера за 2000 ₽/час?
Фрилансер на webasyst.com обычно берёт 1500–3000 ₽/час и тратит 1–2 часа только на онбординг — клонировать репо, развернуть локально, найти нужный файл в wa-apps/shop/themes/. Сама правка — одна строка. SimpleReview знает раскладку файлов Webasyst и открывает PR за секунды. Для сложных задач (1С-интеграция, эквайринг, миграция) — связка с Vibers.
