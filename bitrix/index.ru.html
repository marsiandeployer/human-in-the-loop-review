Что SimpleReview правит на Битриксе

Шаблоны компонентов в /local/templates/[theme]/components/bitrix/
Шаблон сайта в /local/templates/[theme]/header.php, footer.php
Языковые файлы в /local/templates/[theme]/lang/ru/
Init-скрипты в /local/php_interface/init.php
CSS/JS в /local/templates/[theme]/styles.css, script.js
Удаление "Powered by 1С-Битрикс" / редактирование копирайта
Сброс кеша при ошибке "критическая ошибка" (/bitrix/cache/, /bitrix/managed_cache/)
Настройки публичных форм (captcha, valid)

Работает с редакциями "Старт" / "Малый бизнес" / "Бизнес" / "Веб-кластер". Никогда не правит файлы в /bitrix/ (ядро) — только /local/ overrides, чтобы апдейты не сносили правки.

Как работает SimpleReview на сайтах Битрикс

1

Установить расширение Chrome

Auto-detect Битрикса по <meta name="generator" content="Bitrix Site Manager"> + JS-сигнатурам.

2

Подключить Git/SFTP вашего сайта

GitHub/GitLab/Bitbucket/самохост Gitea или прямой SFTP. SimpleReview индексирует /local/ на первом подключении.

3

Кликнуть элемент на витрине

Любой блок, кнопку, копирайт в подвале, элемент компонента. Захватывается DOM + Smarty/PHP source path.

4

Получить Pull Request

Напишите "убрать копирайт Битрикс" или "поправить captcha на форме". Агент правит правильный файл в /local/templates/.../footer.php и открывает PR. Вы мержите или отклоняете.

FAQ

Можно ли править сайт на Битриксе без программиста?
Для 80% задач — да. Шаблоны, языковые файлы, captcha, копирайт. Для интеграции 1С, кастомной оплаты, миграции — нужен живой человек, Vibers.
Как установить SimpleReview на Битрикс-сайт?
Сейчас: Chrome Extension + подключить Git/SFTP. Скоро: нативный модуль на 1С-Битрикс: Маркетплейс (на ревью). Установка через Marketplace → Каталог → SimpleReview → Установить.
Какие редакции Битрикса поддерживаются?
Управление сайтом (Старт / Малый бизнес / Бизнес / Веб-кластер). Все версии 16.0+. Bitrix24 SaaS не поддерживается (нет файлового доступа). Bitrix24 Enterprise on-premise — поддерживается.
Безопасно ли это для прода?
SimpleReview не пушит в основную ветку напрямую. Каждое изменение — PR, который вы ревьюите и мержите. Используйте staging-клон + git workflow.
Чем это лучше фрилансера за 2000 ₽/час?
Скорость и стоимость стартовой задачи. Фрилансеру нужен 1-3 часа просто чтобы клонировать репо и найти файл. SimpleReview уже знает структуру /local/ и открывает PR за секунды.
Что про сложные модули — оплата, 1С-интеграция, миграция Bitrix24 ↔ on-premise?
Это работа для живого специалиста. SimpleReview может подготовить PR-черновик, но финальный merge должен ревьюить человек. Vibers делает именно это — ревью PR-ов перед merge.
